# Client Side

# Installation through bower

```javascript
$ bower install angular-commentWidget
```
Add Module dependency in your main app module

```javascript
angular
    .module('YourMainModule', ['commentWidget']);
```

Use comment-widget directive

```javascript
<ng-comment-widget>
</ng-comment-widget>
```
